# Oyun Mekanikleri

- Ana Bina
Para karşılığı seviye yükseltilebilir
Yağmalama kazancını artırır
- Askeri Bina
Asker enerjisi ile ilişkilidir
Seviye arttıkça enerji ve savaş kapasitesi gelişir
- Araştırma Binası
Kaynak toplama verimini artırır
Geliştikçe daha fazla bonus sağlar

# Asker ve Enerji Sistemi
Askerlerin belirli bir enerji kapasitesi vardır

Kuş avlanabilir
Hazine yağmalanabilir
